# คู่มือการใช้งาน AI Exam Analyzer (ฉบับมือใหม่)

ยินดีต้อนรับสู่ **AI Exam Analyzer**! แอปพลิเคชันนี้จะช่วยคุณวิเคราะห์ข้อสอบ ปรับปรุงคุณภาพ และจัดเก็บข้อสอบลงในคลังข้อสอบโดยใช้พลังของ AI

## 1. สิ่งที่ต้องเตรียม (Prerequisites)

ก่อนเริ่มใช้งาน คุณต้องมั่นใจว่าเครื่องคอมพิวเตอร์ของคุณมีสิ่งเหล่านี้:

1.  **Python**: โปรแกรมภาษา Python (แนะนำเวอร์ชัน 3.8 ขึ้นไป)
    *   ดาวน์โหลดได้ที่: [python.org](https://www.python.org/downloads/)
    *   *ตอนติดตั้ง อย่าลืมติ๊กถูกช่อง "Add Python to PATH"*
2.  **API Key**: กุญแจสำหรับใช้งาน AI (Google Gemini หรือ OpenAI)
    *   **Google Gemini API Key** (ฟรีและแนะนำ): สมัครได้ที่ [aistudio.google.com](https://aistudio.google.com/)

---

## 2. การติดตั้ง (Installation)

ทำตามขั้นตอนทีละสเต็ปดังนี้:

### ขั้นตอนที่ 1: ดาวน์โหลดโปรแกรม
วางโฟลเดอร์โปรเจกต์นี้ไว้ในเครื่องของคุณ (เช่นที่ `Documents/ChatbotExamAnalyzer`)

### ขั้นตอนที่ 2: เปิด Terminal (Command Line)
1.  เข้าไปที่โฟลเดอร์ของโปรเจกต์
2.  คลิกขวาที่พื้นที่ว่าง เลือก **"Open in Terminal"** (หรือพิมพ์ `cmd` ในช่อง Address bar ของโฟลเดอร์แล้วกด Enter)

### ขั้นตอนที่ 3: ติดตั้งตัวช่วย (Dependencies)
พิมพ์คำสั่งต่อไปนี้ลงใน Terminal แล้วกด Enter:

```bash
pip install -r requirements.txt
```

*รอจนกว่าจะติดตั้งเสร็จ (ขึ้นข้อความ Successfully installed...)*

---

## 3. การตั้งค่า (Configuration)

คุณต้องบอกรหัสผ่าน (API Key) ให้โปรแกรมรู้ก่อน:

1.  ในโฟลเดอร์โปรเจกต์ ให้มองหาไฟล์ชื่อ `.env` (ถ้าไม่มีให้สร้างไฟล์ใหม่ชื่อ `.env`)
2.  เปิดไฟล์ด้วย Notepad หรือ Text Editor
3.  ใส่ข้อมูลดังนี้ (เปลี่ยน `YOUR_key` เป็นคีย์จริงๆ ของคุณ):

```env
# เลือกใช้อันใดอันหนึ่ง หรือใส่ทั้งคู่ก็ได้
GOOGLE_API_KEY=AIzaSyDxxxxxxxxxxxxxxxxxxxxxxxxxxxx
OPENAI_API_KEY=sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

4.  บันทึกไฟล์ (Save)

---

## 4. วิธีเริ่มใช้งาน (Running the App)

เมื่อติดตั้งและตั้งค่าเสร็จแล้ว ให้เปิดใช้งานแอปด้วยคำสั่ง:

1.  เปิด Terminal (ในโฟลเดอร์โปรเจกต์)
2.  พิมพ์คำสั่ง:

```bash
streamlit run app.py
```

3.  โปรแกรมจะเปิดหน้าต่างเว็บเบราว์เซอร์ขึ้นมาโดยอัตโนมัติ (ปกติจะอยู่ที่ `http://localhost:8501`)

---

## 5. คู่มือการใช้งานหน้าเว็บ

เมื่อหน้าเว็บโหลดเสร็จแล้ว คุณจะเจอขั้นตอนง่ายๆ 3 ขั้นตอน:

### 1. Upload (อัปโหลดข้อสอบ)
*   ลากไฟล์ข้อสอบของคุณมาวางในกล่อง (รองรับไฟล์ **PDF**, **Word .docx**, หรือ **Text .txt**)
*   ระบบจะทำการ "แกะ" ข้อความออกมาเป็นข้อๆ ให้อัตโนมัติ

### 2. Analyze (วิเคราะห์)
*   เมื่อไฟล์ถูกอ่านแล้ว จะมีปุ่มให้เริ่มวิเคราะห์
*   AI จะทำการตรวจสอบข้อสอบแต่ละข้อตามหลักการวัดผล (Bloom's Taxonomy, ความยากง่าย, ฯลฯ)
*   รอสักครู่... (ความเร็วขึ้นอยู่กับจำนวนข้อ)

### 3. Review & Results (ดูผลลัพธ์)
*   **Dashboard**: ดูภาพรวมว่าข้อสอบดีกี่ข้อ ต้องแก้กี่ข้อ
*   **รายละเอียด**: คลิกดูทีละข้อได้เลย
    *   **สีเขียว**: ข้อสอบคุณภาพดี
    *   **สีแดง**: ควรปรับปรุง (AI จะแนะนำวิธีแก้ให้)
    *   **ปุ่ม Fix**: ให้ AI ช่วยเขียนโจทย์ใหม่ให้ดีขึ้นได้ทันที
    *   **Save to Bank**: บันทึกข้อที่ดีลงใน "คลังข้อสอบ"

## คำถามที่พบบ่อย (FAQ)

*   **Q: ขึ้น Error ว่า `ModuleNotFoundError`?**
    *   A: แปลว่าลืมติดตั้งตัวช่วย ให้กลับไปทำ **ขั้นตอนที่ 3** (`pip install -r requirements.txt`)
*   **Q: วิเคราะห์ไม่ผ่าน / AI ไม่ตอบ?**
    *   A: ตรวจสอบไฟล์ `.env` ว่าใส่ API Key ถูกต้องหรือไม่ และเครดิตของ API ยังไม่หมด
